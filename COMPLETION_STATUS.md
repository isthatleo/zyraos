## 🚀 TO START USING THE SYSTEM

### Quick Start
```bash
# Install dependencies
bun install

# Setup database
bun run db:push

# Start development server
bun run dev

# Visit http://localhost:3000
```

### Create First School
```bash
POST /api/tenant/setup
{
  "name": "Your School Name",
  "slug": "your-school",
  "country": "Your Country",
  "type": "primary|secondary|university",
  "adminEmail": "admin@yourschool.com",
  "adminName": "Admin Name",
  "subscriptionPlanId": "plan_123"
}
```

### Login & Access Dashboards
1. Go to login page
2. Select role/portal
3. Enter credentials
4. Access role-specific dashboard
